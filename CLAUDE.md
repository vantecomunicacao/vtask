### Supabase query patterns

Supabase returns joins as **arrays**, not singular objects — always index `[0]`:
```ts
// ✅ Correct
member.profiles[0]?.full_name

// ✅ Filtering workspace tasks (no direct FK): fetch project IDs first
const { data: projectRows } = await supabase.from('projects').select('id').eq('workspace_id', wid);
const ids = projectRows?.map(p => p.id) ?? [];
supabase.from('tasks').select('*').in('project_id', ids);

// ✅ Join with alias
supabase.from('tasks').select('*, assignee:profiles(*), project:projects(id, name, color)');
```

`setState` inside `useEffect` must use an IIFE async pattern (ESLint `react-hooks/exhaustive-deps`):
```ts
useEffect(() => {
  (async () => {
    const data = await fetchSomething();
    setState(data);
  })();
}, [dep]);
```

### Date handling (`src/lib/dateUtils.ts`)

Supabase stores `due_date` as `TIMESTAMP WITH TIME ZONE` (UTC midnight). Always use:
- `parseDueDate(str)` — parses as local midnight to avoid off-by-one in UTC−3 (Brazil)
- `todayLocalISO()` — returns `yyyy-MM-dd` in local time

Never use `new Date(isoString)` directly for due dates.

### Design system (`src/index.css`)

All tokens are defined as Tailwind v4 `@theme` CSS custom properties — no `tailwind.config.js`. Key token groups:
- **Colors:** `--color-brand`, `--color-surface-{0,1,2,card}`, `--color-text-{primary,secondary,muted}`, `--color-border-{subtle,strong}`
- **Radius:** `--radius-{xs,sm,md,card,lg,pill}`
- **Shadows:** flat design — `--shadow-card: none`, `--shadow-float`, `--shadow-modal`
- **Animations:** `fadeIn`, `slideUp`, `popoverIn/Out`, `staggerIn`, `skeletonPulse`, `taskSlideIn`, `floatingBarIn`
- `prefers-reduced-motion` disables all animations

### UI components (`src/components/ui/`)

- `Button` — variants: `primary | ghost | danger`; sizes: `sm | md | lg | icon`; supports `loading` prop
- `Dialog` — custom modal with focus trap and Escape-to-close
- `Input` — with `error` prop for inline validation messages
- `Select` — custom accessible dropdown with keyboard navigation
- `DatePicker` — standalone + popover variants; shortcuts: Today, Tomorrow, Next week, Next month
- `cn()` in `src/lib/utils.ts` — `clsx` + `tailwind-merge` for conditional classes
